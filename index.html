    Scientific and Tamper-Proof Lottery Mechanism (VRF)

    The OOTD community lottery uses a Verifiable Random Function (VRF) mechanism based on Blockchain Block Hash to ensure absolute fairness and transparency.

    • Eligibility: A random snapshot is taken 15 minutes before the draw. All users holding OOTD Tokens automatically receive an entry number; higher token holdings grant higher winning weight.
    • Random Source: The first block hash generated after 9:30 PM EST every Tuesday is used as the immutable random seed.
    • Anti-Cheating: Results are automatically calculated and announced via smart contract. No one can predict or manipulate the outcome. Winnings are air-dropped directly to user wallets.

    5. OOTD Whitepaper (Core Abstract)

    Abstract

    OOTD is a decentralized fashion ecosystem built on AI and blockchain technology. Through the OOTD Token and NFTs, we aim to solve the core pain points of traditional fashion—poor user experience, inefficient marketing, and data silos—to create a fair, transparent, and efficient digital fashion world.

    Tokenomics (OOTD Token)

    The OOTD Token is the central currency of the OOTD ecosystem. Users acquire OOTD by purchasing value-added services (AI Style Advisor) or contributing high-quality data. OOTD is used by brands to pay for API access and purchase precise advertising slots, while users use it for purchasing virtual clothing NFTs and redeeming exclusive discounts.

    Core Solution

    • B2C Revenue: Users pay for premium AI Style Advisor services (membership or pay-per-use).
    • B2B Revenue: E-commerce giants pay API Access Fees to integrate OOTD’s virtual try-on engine.
    • Future Proofing: Introducing 3D body scanning for hyper-realistic try-ons, solidifying our technological moat.
